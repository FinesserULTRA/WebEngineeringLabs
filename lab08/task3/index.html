IndexedDB Implementation Guide

Objective

Store and retrieve structured data using IndexedDB, a client-side database for storing large amounts of data including objects and files.

Instructions

  1. Enter a task description in the input field
  2. Click "Add Task" or press Enter to store it in IndexedDB
  3. View all tasks displayed in the list below
  4. Toggle task completion status using the Complete button
  5. Delete individual tasks with the Delete button
  6. Use "Delete All Tasks" to clear the entire database

Key Features

  • Stores complex JavaScript objects
  • Auto-incrementing unique IDs for each task
  • Persistent storage with no size limitations (browser-dependent)
  • Asynchronous API for better performance
  • Indexing capabilities for fast queries

Technical Implementation

  • indexedDB.open(name, version) - Open database
  • db.createObjectStore() - Create data store
  • objectStore.add() - Add records
  • objectStore.getAll() - Retrieve all records
  • objectStore.delete() - Remove records
