Cache API Implementation Guide

Objective

Implement offline functionality using Service Workers and Cache API to store static assets for offline access.

Important Note

Service Workers require a web server. They cannot run with the file:// protocol.

Setup Instructions

  1. Option 1: Double-click start-server.bat (requires Python)
  2. Option 2: Use VS Code Live Server extension
  3. Option 3: Run: python -m http.server 8000
  4. Option 4: Run: npx http-server

Testing Procedure

  1. Open DevTools (F12)
  2. Navigate to Application → Service Workers
  3. Verify service worker is registered and activated
  4. Check Application → Cache Storage for cached files
  5. Go to Network tab → Set throttling to "Offline"
  6. Refresh the page - it should still work from cache

Technical Implementation

  • navigator.serviceWorker.register() - Register worker
  • caches.open() - Open cache storage
  • cache.addAll() - Cache multiple files
  • caches.match() - Retrieve from cache
